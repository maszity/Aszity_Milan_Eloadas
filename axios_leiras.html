Feladat

Backend

Fájlok
  • api.php
  • db.php
api.php

HTTP kérések kezelése

  • GET → adatok lekérdezése
  • POST → új sütemény létrehozása
  • PUT → meglévő sütemény módosítása
  • DELETE → sütemény törlése
db.php

Kapcsolódás a MySQL adatbázishoz PDO segítségével

Működés

Adatok lekérése

  • React GET kérést küld az api.php felé
  • PHP lekérdezi az adatbázist (SELECT)
  • JSON formátumban visszaküldi az adatokat

Új sütemény hozzáadása

  • React POST kérést küld
  • PHP beszúrja az adatot (INSERT)
  • Válaszként visszaadja az állapotot

Sütemény módosítása

  • React PUT kérést küld
  • PHP frissíti az adatot (UPDATE)

Sütemény törlése

  • React DELETE kérést küld
  • PHP törli az adatot (DELETE)
