React alapú frontend alkalmazás a CRUD megvalósítására. Az alkalmazás főbb funkciói:

  • Süti lista megjelenítése
  • Hozzáadás / szerkesztés űrlapok külön komponensekben (AddSutiForm, SutiSzerkesztForm)
  • Táblázatos megjelenés SutiTabla komponenssel
  • Minden művelet állapotkezelése a React újrarajzolással

Projekt felépítése

App.jsx

Az alkalmazás alapja, itt vannak definiálva a szükséges változók, az adatbázis adatai itt kerülnek tárolásra egy tömbben. Az alkalmazás logikai metódusai is ide kerültek elhelyezésre (CRUD) amik a szabványos módon a használák a rect useState metódust.

Példa: 

            const [sutemenyLista, setSutemenyLista] = useState(sutemenyek); 
            const addSutemeny = sutemeny => { 
                sutemeny.id = Math.max(...sutemenyLista.map(s => s.id), 0) + 1; 
                setSutemenyLista([...sutemenyLista, sutemeny]); 
            };

AddSutiForm.jsx

React komponens ami megjeleníti a sütemény felvételéhez szükséges űrlapot a hozzá tartozó mezőkkel. A renderelés előtt segéd metódusok lettek létrehozva, amik a useState funkciót használják.

Egy apró, de elegáns megoldással ellenőrzésre kerülnek az adatok, hogy minden szükséges érték meg lett-e adva. Amennyiben nem, a sütemény nem kerül létrehozásra.

Példa: 

            onSubmit={event => { 
                event.preventDefault(); 
                if (!sutemeny.nev || !sutemeny.tipus) return; 
                props.addSuti(sutemeny); 
                setSutemeny({ nev: "", tipus: "", dijazott: 0 }); 
            }}

SutiSzerkesztForm.jsx

Szinte az AddSutiForm.jsx másolata, néhány aprósággal kiegészítve.

Az editRow metódust meghívva (Szerkesztés gomb), a kiválasztott sütemény adatai továbbítva lesznek a SutiSzerkesztForm komponensnek, így a kiválasztott süti értékeit be is tudjuk automatikusan íratni a szerkesztési űrlap mezőibe.
